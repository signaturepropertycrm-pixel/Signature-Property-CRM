# **App Name**: SignatureCRM

## Core Features:

- Property Auto-Title: Automatically generates property titles based on size, unit, and area, such as '5 Marla House in Harbanspura', which simplifies property listing and search.
- Lead Management: Capture and qualify real estate leads (properties & buyers) with automated serial number generation, detailed property and buyer profiles, and status tracking.
- Team Management: Enable Admin, Agent, and Viewer accounts with role-based access control, demo logins, and permission settings to manage team activities and data access.
- Follow-up Automation: Automate follow-ups with reminders and status updates. Log customer information to PostgreSQL and be GDPR compliant.
- Appointment Management: Schedule and manage appointments between agents, buyers, and properties with calendar integration and reminders.
- Performance Analytics: Provide analytics dashboards with KPIs like total properties, total buyers, and monthly revenue/commissions, along with charts showing properties added per month and buyer status distribution.
- Template Copy Generator: Generate personalized templates (property and agent specific) that users can copy to a customer-ready clipboard.

## Style Guidelines:

- Primary color: Deep Blue (#293B5F) to reflect trust and professionalism.
- Background color: Light gray (#E7E8EA) for a clean and modern aesthetic.
- Accent color: Warm Yellow (#FFC857) for highlighting key actions and information.
- Headline font: 'Space Grotesk', sans-serif, for headings and short descriptive text
- Body font: 'Inter', sans-serif, for the majority of content. 
- Use flat design icons to match the simple card-based UI.
- Maintain a clean, card-based layout with rounded corners and subtle shadows, in line with the reference UI.
